# BasicForm
## INTRODUCTION
This is a simple module with a form.</br>
The form contains questions for name, age, birthday and gender.</br>
This form has also data validation.</br>
The module is working on Drupal 8 version.</br>

## INSTALLATION
Installation is usual as most of other Drupal modules.</br>
More information you can find here: https://www.drupal.org/docs/8/extending-drupal-8/installing-drupal-8-modules

## CONFIGURATION
You can find page with form on: "yoursiteurl../myform"
